Blogofile Community

Blogofile is still a small project, but growing rapidly!

  • We already have an active mailing list. Please feel free to join us there if you have any questions about Blogofile.

  • The main issue tracker is also available for you to submit bugs. If you're not sure if it's a bug, then please ask on the mailing list, and we'd be happy to help clarify. If you have feature suggestions, please post them to the discussion list so that we can discuss and flesh them out first.

  • We also hang out in #blogofile on FreeNode, but use the mailing list as your first resource if you expect a timely response :)
